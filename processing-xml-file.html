XML ファイルの処理方法¶

XML 処理の基本¶

  • SAX - Simple API for XML
  • DOM - Document Object Model
  • Pull Parser

多くの場合に DOM を使います。 モダンな実装を求める場合は Pull Parser を使います。 巨大な XML ストリームを扱う場合は SAX を使います。
